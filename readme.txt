== Description ==

Simple Testimonial is plugin for testimonial, where you can add testimonial same as post.


== Installation ==

1. Upload 'simple-testimonial' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the Add Testimonial and create your first Testimonial 
4. You can use shortcode anywhere you want.
5. [show_testimonial] shortcode to show all testimonial.
6. Parameters in Shortcode are:
title,limit,order and testimonial_id
eg. [show_testimonial title='Testimonial' order='DESC' limit=5]
